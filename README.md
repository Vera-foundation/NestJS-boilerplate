## Description

[Nest-quick-starter](https://github.com/nestjs/nest) boost your project startup quickly but keeping full features for development & production.

## Features
- Quick Setup - Setup and configure your application very quickly.
- Follow by NodeJS best practice: [NodeJs best practice](https://github.com/goldbergyoni/nodebestpractices)
- Response formating.
- Seperate environments from development to production.
- JWT Authentication.
- Design for scalable for your projects.
- TDD development.

## Application lifecycle
Middlewares
Guards
Interceptors (before the stream is manipulated)
Pipes
Interceptors (after the stream is manipulated)
Exception filters (if any exception is caught)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
